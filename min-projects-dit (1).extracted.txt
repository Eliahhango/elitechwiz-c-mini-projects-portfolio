MINI PROJECT 1: Student Grade Calculator 
Concepts Covered: 
• Variables 
• Data Types 
• Operators 
• Input/Output (scanf, printf) 
Objective: 
To develop a program that calculates the total, average, and grade of a student. 
Project Description: 
Create a C program that: 
1. Accepts student name and three subject marks. 
2. Calculates total and average. 
3. Displays the student’s grade based on average. 
Grading Criteria: 
Average Grade 
70 – 100 A

Average Grade 
60 – 69 B 
50 – 59 C 
40 – 49 D

Requirements: 
• Use float for marks and average 
• Use arithmetic operators (+, /) 
• Display formatted output 
Sample Output: 
Enter student name: John 
Enter marks for 3 subjects: 75 68 80 
Total = 223 
Average = 74.33 
Grade = A
