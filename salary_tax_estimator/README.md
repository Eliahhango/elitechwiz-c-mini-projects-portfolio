# Salary Tax Estimator

## Description
A mini payroll-style project that calculates gross salary, tax, and net salary.

## Concepts Covered
- Functions and input cleaning
- Arithmetic calculations
- Conditional validation

## Build
```bash
gcc main.c -o salary_tax_estimator
```

## Run
```bash
# Windows
.\salary_tax_estimator.exe
# Linux/macOS
./salary_tax_estimator
```
