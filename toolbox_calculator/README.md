## Description
Menu-driven calculator implementing add, subtract, multiply, and divide operations.

## Concepts Covered
- Switch statements
- Arithmetic operators
- Conditionals
- Input/Output
- Error handling (division by zero)

## Build
```bash
gcc main.c -o toolbox_calculator
```

## Run
```bash
# Windows
.\toolbox_calculator.exe
# Linux/macOS
./toolbox_calculator
```
