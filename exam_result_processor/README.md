# Exam Result Processor

## Description
A mini project to record marks of 5 students and display result statistics.

## Concepts Covered
- Arrays
- Loops
- Conditionals
- Formatted output

## Build
```bash
gcc main.c -o exam_result_processor
```

## Run
```bash
# Windows
.\exam_result_processor.exe
# Linux/macOS
./exam_result_processor
```
