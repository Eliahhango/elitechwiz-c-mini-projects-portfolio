# Hostel Fee Manager

## Description
A mini record manager for storing student hostel fee payments and due balances.

## Concepts Covered
- Structs and arrays
- Functions
- Menu-driven logic
- Formatted reports

## Build
```bash
gcc main.c -o hostel_fee_manager
```

## Run
```bash
# Windows
.\hostel_fee_manager.exe
# Linux/macOS
./hostel_fee_manager
```
