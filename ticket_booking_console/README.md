# Ticket Booking Console

## Description
A mini seat booking system with book, cancel, and view operations.

## Concepts Covered
- Arrays
- Functions
- Menu-driven programs
- Input validation

## Build
```bash
gcc main.c -o ticket_booking_console
```

## Run
```bash
# Windows
.\ticket_booking_console.exe
# Linux/macOS
./ticket_booking_console
```
