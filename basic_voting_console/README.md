# Basic Voting Console

## Description
A mini project for casting votes and viewing vote percentages for three candidates.

## Concepts Covered
- Arrays
- Functions
- Menu-driven program
- Percentage calculation

## Build
```bash
gcc main.c -o basic_voting_console
```

## Run
```bash
# Windows
.\basic_voting_console.exe
# Linux/macOS
./basic_voting_console
```
