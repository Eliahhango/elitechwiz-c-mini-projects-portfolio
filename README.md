## Quick Start
```bash
# 1) Enter any project folder
cd <project_name>

# 2) Compile
gcc main.c -o <project_name>

# 3) Run (Windows)
.\<project_name>.exe

# 4) Run (Linux/macOS)
./<project_name>
```

## Included in Each Project Folder
- `main.c` (source code)
- `README.md` (project-specific details)
- `brief_explanation.doc` (submission brief document)
- `<project_name>.exe` (compiled executable)
