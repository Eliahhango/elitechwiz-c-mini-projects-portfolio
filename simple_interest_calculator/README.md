# Simple Interest Calculator

## Description
A mini project that calculates simple interest and total payable amount.

## Concepts Covered
- Variables and data types
- Arithmetic operators
- Input/Output with scanf and printf
- Conditional validation

## Build
```bash
gcc main.c -o simple_interest_calculator
```

## Run
```bash
# Windows
.\simple_interest_calculator.exe
# Linux/macOS
./simple_interest_calculator
```
