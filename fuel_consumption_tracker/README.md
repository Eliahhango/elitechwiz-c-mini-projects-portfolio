# Fuel Consumption Tracker

## Description
A mini project to record trips and calculate fuel efficiency statistics.

## Concepts Covered
- Structs and arrays
- Functions
- Loops and summaries
- Input validation

## Build
```bash
gcc main.c -o fuel_consumption_tracker
```

## Run
```bash
# Windows
.\fuel_consumption_tracker.exe
# Linux/macOS
./fuel_consumption_tracker
```
