# Number Guess Game

## Description
A mini game where the user guesses a random number with limited attempts.

## Concepts Covered
- Loops
- Conditional statements
- Functions
- Random number generation

## Build
```bash
gcc main.c -o number_guess_game
```

## Run
```bash
# Windows
.\number_guess_game.exe
# Linux/macOS
./number_guess_game
```
