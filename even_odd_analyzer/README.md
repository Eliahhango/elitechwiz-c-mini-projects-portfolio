# Even/Odd Analyzer

## Description
A mini project that analyzes a set of integers into even and odd statistics.

## Concepts Covered
- Arrays/loop-style input processing
- Modulo operator
- Conditionals

## Build
```bash
gcc main.c -o even_odd_analyzer
```

## Run
```bash
# Windows
.\even_odd_analyzer.exe
# Linux/macOS
./even_odd_analyzer
```
