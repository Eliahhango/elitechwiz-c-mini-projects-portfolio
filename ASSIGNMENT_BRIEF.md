# Assignment Brief

## Course
Programming Fundamentals for Technicians (ITT 05114)

## Submission Summary
This submission provides 15 C mini projects. It includes the 4 core assignment-style project categories and additional projects to demonstrate broader understanding of concepts learned in class.

Core categories covered:
- Student grade calculation (event_score_analyzer, student_gradebook_manager)
- Menu-driven calculator (toolbox_calculator)
- Function-based banking (wallet_service_console)
- Array-based student record handling (student_gradebook_manager)

Additional projects extend coverage to inventory, attendance, billing, BMI analysis, unit conversion, library management, expense tracking, payroll, quiz systems, parking fees, and contact management.

## Learning Outcomes Demonstrated
- Correct use of variables, data types, and arithmetic operators
- Strong use of input/output with scanf, fgets, and printf
- Decision-making with if/else and switch
- Iteration with loops for repeated menu workflows
- Function decomposition for modular and reusable logic
- Arrays and structs for storing and processing collections of records
- Defensive input validation and basic runtime error handling
- Formatted report output in tabular form

## Build and Run
For each project folder:
- Build: `gcc main.c -o <project_name>`
- Run on Windows: `.\<project_name>.exe`
- Run on Linux/macOS: `./<project_name>`
