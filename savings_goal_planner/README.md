# Savings Goal Planner

## Description
A mini project that estimates how many months are needed to reach a savings goal.

## Concepts Covered
- Loops
- Arithmetic operations
- Input validation

## Build
```bash
gcc main.c -o savings_goal_planner
```

## Run
```bash
# Windows
.\savings_goal_planner.exe
# Linux/macOS
./savings_goal_planner
```
